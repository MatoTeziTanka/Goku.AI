## üí∞ YIELD FARMING BASICS

### What is Yield Farming?
- Providing liquidity to DeFi protocols to earn rewards
- Similar to earning interest, but with crypto
- Can be staking, lending, or liquidity providing

### Key Concepts

#### 1. Liquidity Pools
- Pairs of tokens locked in smart contract (e.g., ETH/USDT)
- Users trade against the pool
- Liquidity providers (LPs) earn trading fees

#### 2. APY (Annual Percentage Yield)
- Return on investment per year
- **Example**: 50% APY on $1,000 = $500 profit/year

#### 3. Impermanent Loss (CRITICAL RISK!)
- Loss compared to just holding tokens
- Happens when token prices diverge
- **Example**: Provide ETH/USDT at 1:1, ETH 2x ‚Üí you lose 5.7%

#### 4. Smart Contract Risk
- Code bugs can drain funds (rug pulls, exploits)
- Use audited protocols only (Certik, Trail of Bits)

## üîß HOW TO START DeFi (STEP-BY-STEP)

**MetaMask** (most popular):
1. Install browser extension
2. Create wallet (save seed phrase!)
3. Add networks:
   - Ethereum (default)

**Option A: Buy crypto on CEX, withdraw to MetaMask**

- **Polygon Bridge**: bridge.polygon.technology
- **Arbitrum Bridge**: bridge.arbitrum.io

**Cost**: $5-$20 to bridge, then $0.10-$1/tx on L2

### Step 4: Choose a Protocol
**For beginners**:
- **Curve** (stablecoins, low risk)
- **Aave** (lending, simple)
- **PancakeSwap** (BNB Chain, low fees)

### Step 5: Provide Liquidity

2. Connect MetaMask (BNB Chain)
3. Select BNB and BUSD
4. Enter amount (50/50 split)
5. Approve tokens (2 transactions, $0.20 total)
6. Add liquidity
7. Go to "Farms", stake LP tokens
8. Earn CAKE rewards

### Step 6: Monitor & Compound
- Check daily/weekly
- Harvest rewards
- Compound (reinvest rewards)
- Track impermanent loss

## ‚ö†Ô∏è DeFi RISKS (CRITICAL!)

### 1. Smart Contract Risk (Hacks, Bugs)
**Recent examples**:
- Cream Finance: $130M hack (2021)
- Poly Network: $600M hack (2021, returned)
- Ronin Bridge: $600M hack (2022)

**Mitigation**:
- Use audited protocols (Certik, Trail of Bits)
- Check TVL (>$100M = safer)
- Diversify across protocols

### 2. Impermanent Loss
**When it happens**: Token prices diverge

**Example**:
- Start: 1 ETH ($2,000) + 2,000 USDT = $4,000
- ETH 2x to $4,000
- Pool rebalances: 0.707 ETH + 2,828 USDT = $5,656
- If you just held: 1 ETH ($4,000) + 2,000 USDT = $6,000
- **Impermanent loss**: $344 (5.7%)

**Mitigation**:
- Use stablecoin pairs (USDC/USDT)
- Farm correlated assets (ETH/wETH)
- High APY can offset impermanent loss

### 3. Rug Pulls (Scams)
**What**: Developers drain liquidity

**Warning signs**:
- Anonymous team
- No audit
- Low TVL (<$1M)
- Unrealistic APY (>1000%)

**Mitigation**:
- Stick to top 20 protocols
- Check RugDoc, Token Sniffer

### 4. Regulatory Risk
**Concern**: SEC may classify DeFi tokens as securities

**Mitigation**:
- Use decentralized protocols (no KYC)
- Report taxes properly
